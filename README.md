# Contributing to the Archive

Thank you for helping expand this archive!
Anyone is welcome to contribute files that help preserve software, documents, media, or other digital content.

## How to Contribute

The archive accepts contributions through **GitHub Pull Requests**.

### Step 1 — Fork the repository

Click the **Fork** button on the top right of this repository to create your own copy.

### Step 2 — Add your files

Add your files to the appropriate folder in the archive.

Example structure:

/Skin-Packs
/Texture Packs

If a suitable folder does not exist, feel free to create one.

### Step 3 — Commit your changes

Commit the files with a clear message.

Example:

Add: Retro game manual (1998)
Add: Version 1.2 installer for ExampleSoftware

### Step 4 — Create a Pull Request

Open a **Pull Request** to submit your contribution to the archive.

A maintainer will review the submission and merge it if everything looks good.

## Contribution Guidelines

Please follow these guidelines when submitting files:

* Files should be **legal to share**
* Avoid **malware or suspicious executables**
* Provide **original files when possible**
* Use **clear file names**
* Do not upload **extremely large files** unless necessary
* Try to place files in the **correct category**

## File Naming

Use descriptive names when possible.

Good examples:

pack-name_v1.2.zip

Avoid names like:

file1.zip
new.zip
testfile.rar

## Safety

All contributions may be reviewed and scanned before being accepted into the archive.

## Code of Conduct

Be respectful and constructive when contributing.

## Thank You

Every contribution helps preserve digital history.
Thank you for helping build this archive.
